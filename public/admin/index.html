🔐 Mot de passe admin

Le mot de passe est une variable d'environnement Vercel ADMIN_PASSWORD. Pour le modifier, change-la dans le dashboard Vercel puis redéploie.

Settings → Environment Variables → ADMIN_PASSWORD

Supprime toutes les catégories et remet le configurateur à zéro. Cette action est irréversible.
